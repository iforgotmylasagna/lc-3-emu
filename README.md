# LC-3 emulator

# How to build:

    install gcc
      On Fedora: sudo dnf install gcc

    install cmake
      On Fedora: sudo dnf install cmake

    chmod +x compile.sh

    ./compile

# How to run program:

    ./bin/run <file_path>
